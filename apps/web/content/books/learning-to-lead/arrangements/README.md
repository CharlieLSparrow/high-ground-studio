# Arrangements

Arrangement files are playlists of block IDs.

## Purpose

These YAML files define which manuscript blocks belong in which derivative outputs, such as:

- book chapters
- podcast episodes
- public site pages
- courses
- newsletters

## Rule

The arrangement is not the source text.
The arrangement only points at block IDs that live in the manuscript.

## Current state

The files in this directory are starter skeletons only.
They are placeholders for future mapping work after the manuscript is seeded with real block IDs.
