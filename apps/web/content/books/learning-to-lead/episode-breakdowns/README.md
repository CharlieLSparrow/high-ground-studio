# Episode Breakdowns

This folder is a temporary intake layer for episode-structure truth.

Use it for messy, practical working notes about how book material, podcast episodes, Charlie sidebars, clips, research notes, and public status actually map together.

What this folder is for:

- intake notes before formal manuscript or arrangement updates
- partial editorial truth
- clip references
- public/private status notes
- unresolved questions
- side-by-side mapping between book material and episode structure

What this folder is not:

- not the canonical source of prose truth
- not the generated output layer
- not the formal arrangement layer

Working rules:

- the living manuscript remains the source of prose truth
- arrangement YAML remains the formal output map once decisions are incorporated
- breakdown files may stay messy, partial, and unresolved while intake is in progress
- future Codex passes should read these breakdowns first, then update living MDX and arrangement YAML deliberately
- if a prompt references episode material but the actual pasted payload is missing, preserve that fact explicitly in the intake file instead of inventing structure or prose

In short:

- breakdowns = intake truth
- living manuscript = prose truth
- arrangements = formal output order
