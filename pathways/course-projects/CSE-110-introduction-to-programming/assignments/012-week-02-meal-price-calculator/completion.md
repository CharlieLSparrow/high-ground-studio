# Example Completion

This example completes the Meal Price Calculator project with meal prices, guest counts, optional extras, sales tax, tip, total, payment, and change.

## How To Run

```bash
python3 main.py
```

## What It Demonstrates

- converting input to `float` and `int`;
- arithmetic with variables;
- percent calculations;
- formatting money to two decimal places;
- a small conditional for change owed vs. amount still due.
