# Example Completion

This example creates a text adventure with at least three decision points, input validation, branching outcomes, and a clear ending for every path.

## How To Run

```bash
python3 main.py
```

## What It Demonstrates

- `if`, `elif`, and `else`;
- nested decisions;
- a helper function for input validation;
- early `return` for finished branches;
- readable branch structure.
