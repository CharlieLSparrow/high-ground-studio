# Example Completion

This example implements the Clever Stories project as an original mad-lib style story generator.

## How To Run

```bash
python3 main.py
```

Enter the requested words. The program inserts them into a story using an f-string.

## What It Demonstrates

- collecting string input with `input`;
- storing values in variables;
- normalizing one value with `.capitalize()`;
- using an f-string to combine text and variables;
- printing a multi-line result.

## Footnote Notes

The comments in `main.py` use numbered explanatory asides. The goal is to make the code feel annotated by a friendly study partner: a little funny, but mostly useful.
