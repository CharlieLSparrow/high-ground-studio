# Example Completion

This example completes a data analysis project using a small local CSV file.

## How To Run

```bash
python3 main.py
```

Try `2000`, `2010`, or `2020` as the year of interest.

## What It Demonstrates

- reading a CSV file with `csv.DictReader`;
- converting strings to `int` and `float`;
- finding minimum and maximum records;
- filtering rows for a selected year;
- calculating an average.
