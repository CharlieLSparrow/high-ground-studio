# Quipsly brand assets

This folder keeps reusable source material for Quipsly icons, mascot references,
and future visual generation.

## Icon direction

The current production icon comes from the stronger Downloads mascot batch:

- contact sheet option `23`
- centered Quipsly sparrow
- magnifying glass as the research/organization signal
- warm cream app background
- simple enough to survive mobile app icon use
