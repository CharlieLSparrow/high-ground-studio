# Quipsly collective names

Current inside-baseball collective name: **a Marginalia of Quipslys**.

Use this while the product is still mostly for writers, researchers, and other delightful nerds. It reinforces the idea that Quipslys live in the margins: citing, connecting, organizing, and helping without taking the pen away from the human.

## Shortlist

- Marginalia
- Index
- Library
- Schema
- Concern

## Other useful candidates

- Archive
- Cabinet
- Card Catalog

- Ledger
- Quire

- Tangle
- Thread
- Trove

## Product guidance

- Public marketing can say "Quipslys" plainly.
- Internal/operator copy can say "a Marginalia of Quipslys" when the tone allows.
- Avoid making the name too cute in serious trust moments like billing, access, source verification, or data deletion.
- The About page should eventually include the funny names we did not pick.
